# Contributing to War Of Constellations

## Как установить проект

1. **Клонирование репозитория**
   ```bash
   git clone <repository-url>
   ```

2. **Зависимости**
   - Установите **Unity Hub**.
   - Установите версию Unity **2023.1.16f1**.
   - Этот проект использует **Unity Gaming Services (UGS)** для мультиплеера.
     - Убедитесь, что у вас есть доступ к проекту в Unity Dashboard.
     - Перейдите в `Edit > Project Settings > Services` и привяжите проект к вашему Organization ID и Project ID, если потребуется.

3. **Открытие проекта**
   - Откройте Unity Hub.
   - Нажмите "Add" и выберите папку с проектом.
   - Откройте проект (это может занять некоторое время при первом запуске для импорта ассетов).

## Как вносить изменения

1. **Структура проекта**
   - Все скрипты находятся в `Assets/Scripts`.
   - Основные сцены:
     - `Boot`: Начальная сцена загрузки.
     - `MainMenu`: Меню, лобби.
     - `Game`: Основная игровая сцена.

2. **Code Style**
   - Используйте принципы **KISS** (Keep It Simple, Stupid) и **YAGNI** (You Ain't Gonna Need It).
   - Избегайте лишних комментариев, код должен быть самодокументируемым.
   - Имена классов, методов и свойств — PascalCase.
   - Имена приватных полей — _camelCase.

3. **Работа с задачами**
   - Перед началом работы проверьте файл `ToDo.md`, чтобы узнать актуальные задачи.
   - Если вы реализуете заглушку (stub), обновите соответствующий раздел в `ToDo.md` после завершения.

## Как собирать проект

1. Откройте `File > Build Settings`.
2. Убедитесь, что все сцены (`Boot`, `MainMenu`, `Game`) добавлены в список **Scenes In Build**.
3. Выберите платформу (Windows/Mac/Linux).
4. Нажмите **Build** или **Build And Run**.
5. Выберите папку для сборки (например, создайте папку `Builds` в корне репозитория, но не внутри `Assets`).
